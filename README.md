# ANGULAR 1.5 SAMPLE

### Prerequisites
* [NodeJS](https://nodejs.org/en/)

```bash
git clone https://github.com/jbehuet/angular_sample.git   
cd angular_sample
npm i
```

### Development

`npm run dev`
